dev-env.cfg
===========

### install

* vim插件和配置

```bash
git clone git@github.com:liuyanjie/dev-env.cfg.git \
    && git clone https://github.com/VundleVim/Vundle.vim.git ~/.vim/bundle/Vundle.vim \
    && ln -s ${PWD}/dev-env.cfg/vimrc/vimrc ~/.vimrc \
    && vim +PluginInstall +qall
```
